Tilemap Painter + Export (Pro Editor)

Tagline: Map Magic: Paint, Play, Export!

Overview

Tilemap Painter + Export is a professional-grade HTML/JS tilemap editor designed for game developers. Paint, layer, and export tilemaps in seconds with auto-tiling, multi-layer support, and high-res exports compatible with Tiled, Unity, or any engine that supports JSON/PNG tilesets.

Features

Editor Features:

Canvas-based grid with pan & zoom.

Multi-layer system: Background / Mid / Foreground + Collision layer.

Brush tools: Single, Rectangle, Flood Fill, Stamp, Eraser.

Auto-tiling algorithm (Wang or edge-based).

Palette manager: load/save palettes, palette lock.

Layer visibility & opacity controls.

Undo/redo stack (100+ states).


Export & Import:

Bake atlas and pack tiles into PNG.

Export JSON/TMX (Tiled format) with full layer & collision metadata.

Import tilesets with auto-slice.

High-resolution export (x2 / x4 scale).

Save/load project as JSON.

Installation & Usage

1. Open index.html in a modern browser.


2. Use the palette and brush tools to paint layers.


3. Export map via the toolbar to PNG/TMX/JSON.


4. Load saved project JSON to continue editing.
